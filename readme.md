<h2>Бот для подтверждения заявок в телеграм-канал</h2>


<h3>Установка</h3>

1. Введите в консоль сервера команды по порядку:
- `wget https://github.com/ComradeKirik/TGC_Verification_bot/archive/refs/heads/main.zip`
- `unzip main.zip`
- `cd TGC_Verification_bot-main`
2. Введите команду `./start.sh` и следуйте дальнейшим указаниям
